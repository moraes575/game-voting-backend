# Game Voting - Back-end
REST API for a game voting system, being able to view the games, votes and consult the results in a personalized way

## How to  Run
1. Download project dependencies
```
mvn clean install
```
2. Run project
```
mvn spring-boot:run
```

## Technology Stacks
  - Java
  - Spring Boot
  - Spring Data JPA
  - Spring Security
  - Hibernate
  - PostgreSQL
  - Maven
  - JUnit
